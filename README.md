# 🚗 AutoKosten Rob

Persoonlijke PWA voor het bijhouden van autokosten, tankbeurten en APK-datums.

## Auto's
- **Peugeot 406** · 03-FS-HR · 2000 · Automaat
- **Ford Focus** · 91-PF-BL

## Functies
- ⛽ Tankbeurten registreren (Nederland, Duitsland, België)
- 💰 Voordeel berekening t.o.v. Nederland (incl. rijkosten)
- ✅ APK-datums bewaken met waarschuwingen
- 💶 Overige kosten per auto bijhouden
- 📊 Dashboard met totaaloverzicht
- 📱 Installeerbaar als app op telefoon (PWA)
- 💾 Data opgeslagen op je eigen telefoon (localStorage)

## Gebruik

### Online
👉 **https://roblizet.github.io/AutoKosten/**

### Installeren op telefoon
1. Open bovenstaande URL in Chrome (Android) of Safari (iPhone)
2. Android: tik op de drie puntjes → *Toevoegen aan startscherm*
3. iPhone: tik op het deel-icoontje → *Zet op beginscherm*

## GitHub Pages activeren
1. Ga naar **Settings** → **Pages**
2. Source: `Deploy from a branch`
3. Branch: `main` / `root`
4. Klik **Save** — na ~1 minuut is de app live

---
*Data blijft lokaal op je telefoon. Gebruik de export-knop in Instellingen voor een backup.*
